# 小程序图标说明

请将以下图标文件放入 `assets/icons/` 目录：

## TabBar 图标（需要 81x81 像素的 PNG 图片）

| 文件名 | 用途 | 建议颜色 |
|--------|------|----------|
| today.png | 今日（未选中） | 灰色 #999 |
| today-active.png | 今日（选中） | 蓝色 #4A90D9 |
| habit.png | 习惯（未选中） | 灰色 #999 |
| habit-active.png | 习惯（选中） | 蓝色 #4A90D9 |
| stats.png | 统计（未选中） | 灰色 #999 |
| stats-active.png | 统计（选中） | 蓝色 #4A90D9 |
| profile.png | 我的（未选中） | 灰色 #999 |
| profile-active.png | 我的（选中） | 蓝色 #4A90D9 |

## 临时解决方案

在开发阶段，可以先使用以下方式：
1. 使用 iconfont 在线生成图标
2. 使用设计工具（如 Figma、Sketch）导出
3. 使用第三方图标库（如 iconfont.cn）

## 图标设计建议

- 简洁明了，避免过多细节
- 使用统一的视觉风格
- 选中状态使用品牌色（#4A90D9）
- 未选中状态使用灰色（#999999）
