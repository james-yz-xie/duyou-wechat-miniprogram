## 项目概述

**督友(DuYou)** 是一款微信小程序，专注于帮助用户追踪和管理日常习惯。

**核心价值**：
- 📋 创建和管理多个习惯
- ✅ 日常打卡记录和追踪
- 📊 数据统计和可视化
- 🔔 智能提醒功能
- 👤 个人资料管理

**技术亮点**：
- 混合云架构，支持离线模式
- API优先 + 本地缓存的双模式运行
- Spring Boot后端，微信云托管部署
- 自动降级机制，网络异常时无缝切换

### 架构设计原则

| 原则 | 实现方式 |
|------|--------|
| **用户至上** | 离线可用，网络异常自动降级 |
| **性能优先** | API + 本地缓存，减少网络请求 |
| **数据安全** | 微信认证 + 服务端授权 + HTTPS加密 |
| **代码分离** | UI / 存储 / 网络 / 工具函数分层清晰 |
| **可扩展性** | 易于添加新功能和数据类型 |

### 5.2 连续打卡计算逻辑

```javascript
function calculateStreak(habitId) {
  const checkIns = getAllCheckIns();
  const habitCheckIns = checkIns[habitId] || {};
  const dates = Object.keys(habitCheckIns).sort().reverse();
  
  if (dates.length === 0) return 0;
  
  const today = new Date();
  today.setHours(0, 0, 0, 0);
  
  let streak = 0;
  let currentDate = today;
  
  // 从今天或昨天开始计算
  if (dates.includes(todayStr)) {
    streak = 1;
    currentDate = new Date(currentDate.getTime() - 86400000);
  } else if (dates.includes(yesterdayStr)) {
    streak = 1;
    currentDate = new Date(currentDate.getTime() - 86400000 * 2);
  } else {
    return 0;
  }
  
  // 向前遍历，统计连续天数
  while (true) {
    const dateStr = formatDate(currentDate);
    if (dates.includes(dateStr)) {
      streak++;
      currentDate = new Date(currentDate.getTime() - 86400000);
    } else {
      break;
    }
  }
  
  return streak;
}
```

### 8.2 存储优化

```javascript
// 使用本地缓存减少 API 调用
async function getHabits() {
  if (useApi) {
    try {
      const habits = await api.getHabits();
      wx.setStorageSync('habits', habits);  // 缓存
      return habits;
    } catch (error) {
      return wx.getStorageSync('habits') || [];  // 回退
    }
  }
  return wx.getStorageSync('habits') || [];
}
```

## 10. 总结

督友小程序采用了**分层架构 + 双模式运行**的设计理念：

1. **清晰的层次结构**：页面层 → 存储层 → 网络层，职责分明
2. **健壮的双模式**：在线时使用云端 API，离线时自动降级到本地存储
3. **微信原生集成**：充分利用微信登录、云托管、订阅消息等能力
4. **数据安全第一**：敏感数据存储在服务端，本地仅缓存必要数据

这种架构既保证了功能的完整性，又提供了良好的用户体验，即使在网络不稳定的情况下也能正常使用。
